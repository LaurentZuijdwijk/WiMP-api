Unofficial Tidal/WiMP API
====================
This library can currently:
 - Log in to the service
 - Get a list of albums
 - Get tracks in an album
 - Get playlists
 - Get the streaming url for a track
 - Play a track
 - Get featured playlists
 - Much more
